# GetX Clean Starter Template

A starter template for Flutter **3.41.6** with:

- GetX route management, state management and dependency injection
- Clean-ish scalable folder structure
- Reusable widgets
- Splash, onboarding, login, register, bottom navigation holder
- Theme, strings, colors, image paths, validators, storage service
- API client scaffold using Dio

## Run

```bash
flutter pub get
flutter run
```

## Suggested next steps

1. Replace dummy auth service with real API calls.
2. Add feature modules as your app grows.
3. Put real assets in `assets/images` and `assets/icons`.
4. Add localization and environment config if needed.
